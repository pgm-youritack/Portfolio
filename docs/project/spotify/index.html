spotify

beschrijving

Het is een zelf gemaakte applicatie waarbij je kan registeren als een administrator,writer of reader.
Iedereen kan een profiel foto toevoegen het kan een jpg,png,svg of een gif zijn.
Als administrator kan je artiesten toevoegen/aanpassen, albums toevoegen/aanpassen en ook voor songs.
Als writer kan je artiesten aanpassen, albums aanpassen en ook voor songs.
Als reader kan je alles gewoon bekijken maar geen toepassing voor toevoegen en aanpassen.
